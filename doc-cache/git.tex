\chapter{Git~学习}

\section{Git~的安装}
在~Ubuntu~下打开终端（ctrl + alt + t），并在终端中输入：
 \begin{code}
 sudo apt-get install git
 \end{code}
 
\section{Git~的一些初始配置}
可以根据自己的具体需要对~git~作一些初始的设置
\begin{code}
git config --global user.name “Your Name”
git config --global user.email “you@example.com”
\end{code}
当然还有很多其他的设置，可以通过~man git~查看
